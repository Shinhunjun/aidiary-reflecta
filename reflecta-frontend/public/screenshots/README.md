## How to Take Screenshots

### For macOS:
1. **Full screen**: `Cmd + Shift + 3`
2. **Selected area**: `Cmd + Shift + 4` (then drag to select)
3. **Window**: `Cmd + Shift + 4`, then press `Space`, then click window

### For Windows:
1. **Full screen**: `PrtScn`
2. **Selected area**: `Win + Shift + S`
3. **Window**: `Alt + PrtScn`

### Browser DevTools (Recommended for consistent sizing):
1. Open Chrome DevTools (F12)
2. Click the device toolbar icon (Cmd/Ctrl + Shift + M)
3. Set custom dimensions (e.g., 1400×1000)
4. Take screenshot using browser's capture tool or OS shortcuts

## Editing Screenshots (Optional)

If needed, you can:
- Crop to focus on key areas

## File Naming Convention

- Use **lowercase** and **hyphens** (e.g., `mandalart-grid.png`)
- Use **PNG** format for best quality
- Keep files under 2MB if possible (compress if needed)
